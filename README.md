# MofuTenshi

Art-focused e-commerce storefront for a single brand — figures, handcrafts, art prints, stickers. A lightweight, self-hosted alternative to Shopify.

**Stack:** Next.js (App Router) · TypeScript · PostgreSQL · Prisma · Auth.js · Tailwind CSS · Radix UI · Cloudflare R2 · Docker/Coolify on VPS
